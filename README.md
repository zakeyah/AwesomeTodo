# AwesomeTodo

+ Phone App was made by react native
+ It is a simple notes app to keep your notes in and avoid losing your mind

* React Native
* [Expo](https://m.apkpure.com/ar/expo/host.exp.exponent) 

# Running
* ``npm install --global expo-cli``
* `npm i `
* `expo start`
